Steps to Apply for New Passport

  1. Register/login on the official passport website.
  2. Fill the application form, upload required documents.
  3. Pay the fee.
  4. Schedule appointment.
  5. Visit Passport Seva Kendra (PSK) for verification.
  6. Track application status.

If you have any questions or concerns about the passport application process, please Contact us. We are here to help you.
